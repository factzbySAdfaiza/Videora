=====================================================
         VIDEO STUDIO - QUICK START GUIDE
=====================================================

FIRST TIME SETUP (only once):
1. Open this folder
2. Click on the address bar, type "cmd" and press Enter
3. In the black window, type: npm install
4. Wait for it to finish (1-2 minutes)

CREATE A VIDEO (every time you want a new one):
1. Go to LM Arena (lmarena.ai) and use Claude
2. Paste the PROMPT we gave you and describe your animation
3. Copy the code that the AI gives you
4. Open the file "src/MyVideo.tsx" with Notepad
5. Delete everything and paste the new code. Save.
6. In CMD type: npm run build
7. Your video will be in the "out" folder

DONE! Repeat the "CREATE A VIDEO" steps to make more.
